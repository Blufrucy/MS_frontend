# 测试账号说明

## 🔐 如何创建测试账号

### 方法1：通过注册页面（推荐）

1. 访问注册页面：http://localhost:3000/register

2. 填写信息：
   - **昵称**：随便填（例如：测试用户）
   - **邮箱**：随便填（例如：test@example.com）
   - **验证码**：随便填（例如：123456）- 已禁用验证
   - **密码**：至少6位（例如：123456）
   - **手机号**：选填

3. 点击"注册"

4. **自动登录**：注册成功后会自动使用邮箱登录，直接跳转到首页

5. **说明**：系统会自动生成用户名（格式为数字，例如：10001），但你不需要记住它，可以直接使用邮箱或手机号登录

### 方法2：直接使用数据库中的账号

如果数据库中已有用户数据，可以直接使用：

```sql
-- 查询所有用户
SELECT id, username, nickname, email, phone FROM user;
```

## 🎯 登录说明

### 登录方式

系统支持3种登录方式（任选其一）：

1. **用户名登录**：使用系统生成的用户名（例如：10001）
2. **邮箱登录**：使用注册时填写的邮箱
3. **手机号登录**：使用注册时填写的手机号

### 登录步骤

1. 访问登录页面：http://localhost:3000/login

2. 输入：
   - **用户名**：邮箱（推荐）/ 手机号 / 系统生成的用户名
   - **密码**：注册时设置的密码

3. 点击"登录"

**提示**：推荐使用邮箱登录，因为邮箱是你自己填写的，容易记住

## ❓ 常见问题

### Q1：忘记了系统生成的用户名怎么办？

**方法A：查看数据库**
```sql
SELECT username, nickname, email FROM user WHERE email = 'your-email@example.com';
```

**方法B：使用邮箱或手机号登录**
- 直接在用户名框输入邮箱或手机号即可

### Q2：提示"账号或密码错误"

**可能原因：**
1. 用户名/邮箱/手机号输入错误
2. 密码输入错误
3. 该账号不存在

**解决方案：**
1. 检查输入是否正确
2. 尝试使用邮箱或手机号登录
3. 重新注册一个新账号

### Q3：注册时提示"该邮箱可能已被注册"

**原因：** 该邮箱已经被使用

**解决方案：**
1. 使用其他邮箱注册
2. 或者直接使用该邮箱登录

### Q4：如何查看我的用户名？

注册成功后，系统会自动生成用户名。有两种方式查看：

**方式1：登录后查看**
- 登录成功后，右上角会显示你的昵称
- 点击头像下拉菜单 → 个人中心，可以看到用户名

**方式2：查询数据库**
```sql
SELECT username, nickname, email FROM user ORDER BY id DESC LIMIT 1;
```
